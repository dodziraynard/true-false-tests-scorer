# True-False Test Scorer

This is a simple app for scoring true/false based tests.

# Language

Python 3

# Installing

```bash
git clone https://github.com/dodziraynard/true-false-tests-scorer.git
```

# Running

```bash
cd .\true-false-tests-scorer
python .\app.py
```
